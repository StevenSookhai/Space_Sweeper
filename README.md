# SpaceSweeper
# Background
What happens when you happen to get on the bad side of the Space Mafia? Wellâ€¦.you either try to survive or be eliminated! Space Sweeper is a space themed game where you are flying in space fighting your way through against enemies sent to eliminate you.
# Functionality & MVPs
In Space Sweeper, users will be able to:
- fly a spaceship through space using using the wasd keys
- shoot enemies using the space key
- play through different levels
- collect items that will add extra modification to their ship and upgrade ship
- be able to start/pause game

In addition, this project will include:
- An About modal describing the background and rules of the game
- A production README

# Technologies, Libraries, APIs
This project will be implemented with the following technologies:
- The Canvas API to render the game board
- Webpack bundle and transpile the source JavaScript code
- npm to manage project dependencies
- HTML & CSS
